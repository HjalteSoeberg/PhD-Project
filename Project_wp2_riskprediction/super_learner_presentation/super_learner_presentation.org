* Introduction

*** Problem
We want to use variables ,$W$, from the registers to predict the risk of getting diagnosed with type 2 diabetes, $Y$ ($Y = 1$ means the person has T2D). To do this we need to find some function $\psi_0(W) = P(Y = 1)$. \newline
\newline
\pause
How do we find this function?


*** Super leaner
Assume we have the data $X_i = (W_i,Y_i)$, $i = 1,...,n$, where $W_i$ is a p-dimensional set of covariates and $Y_i$ is the outcome of interest.\pause  Also assume we are given a library of learners $\mathcal{L} = (\psi_1(W),...,\psi_K(W))$ containing $K$ learners.\pause The super learner algorithm will then find the function that minimizes the expectation of some loss- or score-function:
#+begin_export latex
\begin{align*}
\psi_0(W) = \arg\min_{\psi\in\mathcal{L}}E[L(X,\psi(W))],
\end{align*}
#+end_export
where the loss function $L$ is usually the squared loss $L = (Y-\psi(W))^2$. This is done by using cross validation.

*** The discrete super learner
Picks the best performing model contained in the library.

- Pros:
  + All theoretical results shown by van der Laan and Polley holds \pause
  + Can perform aswell as the oracle estimator \pause
  + Performs well in practise\pause
  + Low runtime\pause

- Cons:
  + Not as fancy as the ensemble learner


*** The ensemble super learner
Combines all the learners in the library to create a new "super" learner.

- Pros:
  + Fancy \pause
  + Usually outperforms the discrete superlearner during training \pause

- Cons:
  + No theoretical results \pause
  + Can never be the best learner \pause
  + Doesnt perform well in practise \pause
  + Crazy bad runtime

*** Performance measures: AUC
Measures discrimination, i.e. how good is the model to discriminate between a person who gets diabetes and a person who doesnt. The AUC for a model is the probability of the model, given a random person who gets T2D and a random person who doesnt, to give the T2D person a higher risk.

*** Performance measures: Brier-score
Measure overall performance of a prediction model, and can be used as a standalone measure as it is a theoretical proper measure.
#+begin_export latex
\begin{align*}
BS = \frac{1}{n}\sum_{i = 1}^n(\psi_0(W_i)-Y_i)^2
\end{align*}
#+end_export



*** Calibration plots
Calibration in the large refers to if the model average predicted probability is the same as the observed average probability of the outcome.
\newline
Calibration in the small refers to if the predicted probabilites match the observed probabilites, i.e. 25% of the people who get a prediction of 25% should get the event.
